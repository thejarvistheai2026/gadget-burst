# Burst + Churn Tracker

## Quick Start

```bash
# Start auto-sync mode (watches files, syncs automatically)
ggt dev

# Or manual push
ggt push --allow-unknown-directory
```

## Development

Open `ggt dev` in a terminal tab, then edit files in VS Code/Cursor. Changes sync automatically to Gadget.

Full documentation: See `../burst-churn-addon/README.md`
